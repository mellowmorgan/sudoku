# Sudoku Game

#### By Morgan Waites

## Technologies Used

* Javascript
* HTML/CSS/Bootstrap
* Jest

## Description

Webpage has one sudoku template you can try to win.

## Setup/Installation Requirements

* git clone this repo
* navigate to top-level of directory
* npm install
* npm run start
* npm test to run tests
